# Pendiente

$m = \frac{\vartriangle y}{\vartriangle x}$
pendiente perpendicular ($m_2$) dada una pendiente ($m_1$): $m_2 = -\frac{1}{m_1}$
encontrar la función dada una pendiente y un punto: $y - y_1 = m(x - x_1)$
distancia entre un punto y una pendiente dada la función en su forma general ($Ax + By + C$):
$d = \frac{|Ax_0 + By_0 + C|}{\sqrt{A^2 + B^2}}$
donde:
$Ax_0$ es el valor que multiplica a x en la forma general por el punto x
$By_0$ es el valor que multiplica a y en la forma general por el punto y
$C$ el valor de C en la forma general
A el valor que multiplica a x en la forma general
B el valor que multiplica a y en la forma general
# Cónicas

**Formula general**: $Ax^2 + Bxy + Cy^2 + Dx + Ey +  F = 0$

#### Identificación de cónicas

Si $A \text{ o } C = 0$  es una parábola
Si $\pm A = \pm C$, (mismo valor, mismo signo) es una circunferencia
Si $\pm A \neq \pm C$, (diferente valor, mismo signo) es una elipse
Si $\pm A \text{ y } \mp C$, (signos diferentes) es una hipérbola

#### Discriminante $I = B^2 - 4AC$

Si $I = 0$ es una parábola
Si $I < 0$ es una elipse (Si además $A = C y B = 0$, es una circunferencia)
Si $I > 0$ es una hipérbola

#### Circunferencia

Centro: $C(h, k)$
Formula: $(x - h)^2 + (y - k)^2 = r^2$

#### Parábola

Vértice: $V(h, k)$
Lado recto: $Lr = 4p$
Formulas:

- Vertical: $(x - h)^2 = 4p(y - k)$, si p es positivo U (sube), si es negativo ∩ (baja)
- Horizontal: $(y - k)^2 = 4p(x - h)$, si p es positivo $\subset$ (derecha), si es negativo $\supset$ (izquierda)

#### Elipse

Centro: $C(h, k)$
$a > b$
$c = \sqrt(a^2 - b^2)$ Pitágoras pero la hipotenusa es a
Excentricidad: $e = \frac{c}{a}$
Eje mayor: $2a$
Eje menor: $2b$
Lado recto: $Lr = \frac{2b^2}{a}$
Formulas:
- Vertical: $\frac{(x - h)^2}{b^2} + \frac{(y - k)^2}{a^2} = 1$
- Horizontal: $\frac{(x - h)^2}{a^2} + \frac{(y - k)^2}{b^2} = 1$

#### Hipérbola

Centro: $C(h, k)$
Lado recto: $Lr = \frac{2b^2}{a}$
$a = C - V$
$b = C - \text{eje conjugado}$
$c = C - F$
$c = \sqrt{a^2 + b^2}$ Pitágoras
Excentricidad: $e = \frac{c}{a}$
Formulas:
- Vertical: $\frac{(y - k)^2}{a^2} - \frac{(x - h)^2}{b^2} = 1$
- Horizontal: $\frac{(x - h)^2}{a^2} - \frac{(y - k)^2}{b^2} = 1$

# Derivadas

Sea $u=u(x)$, $v=v(x)$ y $c$ una constante.

1. $y = c \Rightarrow y' = 0$

2. $y = x \Rightarrow y' = 1$

3. $y = cx \Rightarrow y' = c$

4. $y = x^n \Rightarrow y' = n x^{\,n-1}$

5. $y = c\,u \Rightarrow y' = c\,u'$

6. $y = \pm u \pm v \pm \cdots \Rightarrow y' = \pm u' \pm v' \pm \cdots$

7. $y = u^n \Rightarrow y' = n u^{\,n-1} u'$

8. $y = u v \Rightarrow y' = u v' + v u'$

9. $y = \dfrac{u}{v} \Rightarrow y' = \dfrac{v u' - u v'}{v^2}$
