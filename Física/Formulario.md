### MRU

$V = \frac{d}{t}$
$R = \frac{d}{t}$

### MRUA

$a = \frac{V_F - V_0}{t}$
$d = \frac{V_F + V_0}{2} t$
$d = V_0t + \frac{at^2}{2}$
$V_F^2 = V_0^2 + 2ad$
$V_F = V_0 + at$

R = rapidez (magnitud)
V = velocidad (magnitud vectorial)
d = distancia, también se puede representar con x
t = tiempo
a = aceleración
$V_F$ = Velocidad final
$V_0$ = Velocidad inicial
